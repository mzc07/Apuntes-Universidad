### Consultar los siguientes conceptos
- Sistema operativo (Ejemplos contemporaneos e historicos)
- Protocolos de red (Y sus tipos)
- Archivos y sistemas de archivos
### Solucion
- Un sistema operativo es el conjunto de programas de un sistema informatico que gestiona los recursos del hardware y permite proveer servicios a los programas de aplicaciones. 
1. **GNU/Linux:** Sistema operativo de codigo abierto monolitico. Famoso en el ambito de desarrollo por su seguridad, personalizacion y su filosofia de codigo abierto.
2. **Windows:** Sistema operativo desarrollado por Microsoft. Es el sistema operativo mas famoso del mundo.
3. **MacOS:** Sistema operativo desarrollado por Apple. Usado en el ambito de diseño grafico.
4. **Solaris:** Sistema operativo desarrollado originalmente por Sun Microsystems enfocado en las aplicaciones de Java.
5. **UNIX:** Sistema operativo antiguo desarrollado por Ken Thompson y Dennis Ritchie. Es la base de muchos de los sistemas operativos hoy en dia.
6. **TempleOS:** Sistema operativo desarrollado por Terry Davis. Buscaba crear un sistema operativo con tematica religiosa.

- Los protocolos de red son el conjunto de reglas estandarizadas que permiten a los dispositivos informaticos comunicarse para intercambiar informacion de manera coherente.
1. **Comunicacion e Internet:** IP para direccionar y enrutar a traves de redes. TCP para garantizar la entrega de datos confiable. UDP para transmision rapida sin conexion.
2. **Aplicacion:** HTTP/HTTPS transfiere paginas web, SMTP envio de correo electronico, DNS traduce nombres de dominio a direccion IP, DHCP asigna direcciones IP automaticamente.
3. **Seguridad**: SSL/TLS cifra datos en transito, IPSec protege la comunicacion IP.
4. **Administracion:** SNMP supervisa y mantiene redes, ICMP gestiona mensajes de error y diagnostico.

- Un archivo es una coleccion de datos almacenado en un sistema informatico.
1. **NTFS:** Estandar de Windows que soporta grandes archivos y seguridad avanzada.
2. **FAT32:** Compatible con diversos SO pero limita el tamaño de archivo a 4GB.
3. **exFAT:** Compatible con Windows y Mac usado en memorias externas.
4. **APFS:** Optimizado para discos de estado solido (SSD)
5. **ext4:**  Comun en Linux por su eficiencia y confiabilidad.
6. **ReFS:** Sistema resiliente  de Microsoft usado por su escalabilidad y rendimiento en servidores.
