### Consultar los siguientes conceptos
- Sistema operativo (Ejemplos contemporáneos e históricos)
- Protocolos de red (Y sus tipos)
- Archivos y sistemas de archivos
### Solución
- Un sistema operativo es el conjunto de programas de un sistema informático que gestiona los recursos del hardware y permite proveer servicios a los programas de aplicación de software. 
1. **GNU/Linux:** Sistema operativo de código abierto monolítico modular. Famoso en el ámbito de desarrollo por su seguridad, personalización y su filosofía de código abierto.
2. **Windows:** Sistema operativo desarrollado por Microsoft. Es el sistema operativo mas famoso del mundo.
3. **MacOS:** Sistema operativo desarrollado por Apple. Usado en el ámbito de diseño grafico.
4. **Solaris:** Sistema operativo desarrollado originalmente por Sun Microsystems enfocado en las aplicaciones de Java.
5. **UNIX:** Sistema operativo antiguo desarrollado por Ken Thompson y Dennis Ritchie. Es la base de muchos de los sistemas operativos hoy en día.
6. **TempleOS:** Sistema operativo desarrollado por Terry Davis. Buscaba crear un sistema operativo con temática religiosa.

- Los protocolos de red son el conjunto de reglas estandarizadas que permiten a los dispositivos informáticos comunicarse para intercambiar información de manera coherente.
1. **Comunicación e Internet:** IP para direccionar y enrutar a través de redes. TCP para garantizar la entrega de datos confiable. UDP para transmisión rápida sin conexión.
2. **Aplicación:** HTTP/HTTPS transfiere paginas web, SMTP envió de correo electrónico, DNS traduce nombres de dominio a dirección IP, DHCP asigna direcciones IP automáticamente.
3. **Seguridad**: SSL/TLS cifra datos en transito, IPSec protege la comunicación IP.
4. **Administración:** SNMP supervisa y mantiene redes, ICMP gestiona mensajes de error y diagnostico.

- Un archivo es una colección de datos almacenado en un sistema informático.
1. **NTFS:** Estándar de Windows que soporta grandes archivos y seguridad avanzada.
2. **FAT32:** Compatible con diversos SO pero limita el tamaño de archivo a 4GB.
3. **exFAT:** Compatible con Windows y Mac usado en memorias externas.
4. **APFS:** Optimizado para discos de estado solido (SSD)
5. **ext4:**  Común en Linux por su eficiencia y confiabilidad.
6. **ReFS:** Sistema resiliente  de Microsoft usado por su escalabilidad y rendimiento en servidores.
